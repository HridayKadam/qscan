# 📦 Qscan – Scan Smarter, Shop Easier

**Qscan** is a mobile-first web app that lets users **scan barcodes or product images** to view key details like **product name, MRP, and expiry date** — no more squinting at labels in grocery stores.

Built using vanilla HTML, CSS, and JavaScript, with powerful libraries for barcode and image recognition.

---

## ✨ Features

- 🔍 **Barcode Scan** – Instantly decode product barcodes using your camera
- 🖼️ **Image Scan (OCR)** – Upload or snap a photo to extract text
- 🧠 **Smart Parsing** – Auto-detect product name, MRP, and expiry
- 🕹️ **Mobile-Optimized UI** – Built with a clean, responsive layout
- 💾 **Local Only** – All data processed client-side for privacy

---

## 🛠️ Tech Stack

- HTML, CSS, JavaScript
- [`html5-qrcode`](https://github.com/mebjas/html5-qrcode) – barcode scanner
- [`Tesseract.js`](https://github.com/naptha/tesseract.js) – OCR engine
- Hosted on [Vercel](https://vercel.com) (optional)
